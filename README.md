# Personal-zima - Apps
Recreated Zima Apps for Perosnal Use

## Apps List

### Glance
[Glance](https://github.com/glanceapp/glance) A lightweight, highly customizable dashboard that displays your feeds in a beautiful, streamlined interface. 
** Note: Don't forget to copy the glance.yml to config folder.

### Metube
[Metube](https://github.com/alexta69/metube) is a web GUI for youtube-dl (using the yt-dlp fork) with playlist support. Allows you to download videos from YouTube and dozens of other sites.

### Newt
[Newt](https://github.com/fosrl/newt) is a fully user space WireGuard tunnel client and TCP/UDP proxy, designed to securely expose private resources controlled by Pangolin. By using Newt, you don't need to manage complex WireGuard tunnels and NATing.

### Tubesync
[Tubesync](https://github.com/meeb/tubesync) is a PVR (personal video recorder) for YouTube. Or, like Sonarr but for YouTube (with a built-in download client). It is designed to synchronize channels and playlists from YouTube to local directories and update your media server once media is downloaded.
